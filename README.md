#Projex Mobile
Projex Mobile é uma solução intuitiva desenvolvida para a visualização e acompanhamento de solicitações de tarefas no âmbito de associações de municípios integrados. O aplicativo funciona como um painel de controle centralizado, permitindo que gestores monitorem o ciclo de vida completo das demandas municipais de forma ágil e transparente.

#Sobre o Projeto
O objetivo principal do Projex Mobile é facilitar o controle do andamento das atividades solicitadas por prefeituras e órgãos públicos. A ferramenta é essencial para Administradores, Secretários e Prefeitos, proporcionando uma visão macro e micro da gestão pública em tempo real.

#Status de Tarefas
O aplicativo permite filtrar e visualizar tarefas organizadas pelos seguintes status:

Em andamento: Tarefas que estão sendo executadas no momento.

Paralisadas: Atividades que aguardam alguma resolução para prosseguir.

Finalizadas: Tarefas concluídas pela equipe operacional, aguardando revisão.

Concluídas: Ciclo completo e documentado.

#Tecnologias Utilizadas
Este projeto foi construído utilizando tecnologias modernas para garantir performance e escalabilidade:

Dart & Flutter: Para o desenvolvimento da interface e lógica do aplicativo mobile.

API Rest: Para integração com o sistema de gestão central das associações.

JSON: Para intercâmbio de dados.
